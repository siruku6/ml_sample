# ML_Sample

# 1. Requirement

You need to setup these and to enable `docker-compose` to run.

- docker
- docker-compose

## Other Middlewares

- pipenv
- postgresql

**\*You don't have to setup these Middlewares in your OS, if you build your environment with docker and docker-compose !!**
